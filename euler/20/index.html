Problem Statement

Find the sum of the digits in \(N!\).

My Algorithm

With Python, an \(O(1)\) solution is simple. We compute \(n!\), convert it to a string, and sum the digits.

from math import factorial

def digit_sum(n):
	return sum([int(x) for x in str(n)])

T = int(input())
for _ in range(T):
	N = int(input())
	print(digit_sum(factorial(N)))
