Problem Statement

Considering natural numbers \(a,b \le N\), which number of the form \(a^b\) has the maximum digital sum?

My Algorithm

For each \(a,b \le N\), we simply compute \(a^b\) in \(\log b\) time and store its digital sum if it is maximal so far. This solution has time complexity \(O(N^2\log N)\).

def dig_sum(n):
	return sum([int(x) for x in str(n)])

N = int(input())
print(max([dig_sum(a**b) for a in range(1,N) for b in range(1,N)]))
