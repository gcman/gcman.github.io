Problem Statement

The sum of all primes below 10 is \(2+3+5+7 = 17\). Find the sum of all primes at most \(N\).

My Algorithm

We use a Sieve of Eratosthenes to compute the primes up to the maximum possible value of \(N\). We then construct a prefix sum array in \(O(\frac{N}{\log N})\) time. Using a binary search, we find the index of largest prime less than \(N\) in \(O(\log N - \log \log N)\) time. In \(O(1)\) time, we look up the corresponding partial sum of prime numbers. Our solution has time complexity \(O(n\log\log n + \frac{n}{\log n} + T\log n)\), where \(T\) is the number of queries.

def bs(arr, l, r, x):
	while l <= r:
		mid = l + (r - l)//2;
		if arr[mid] == x:
			return mid
		elif arr[mid] < x:
			l = mid + 1
		else:
			r = mid - 1
	return r

# Sieve of Eratosthenes
def primes(n):
	primes = []
	sieve = [True] * (n + 1)
	for p in range(2, n + 1):
		if sieve[p]:
			primes.append(p)
			for i in range(p * p, n + 1, p):
				sieve[i] = False
	return primes

PRIMES = primes(int(2e6))
# Construct the prefix sum array
P = [0]
count = 0
for p in PRIMES:
	count += p
	P.append(count)
del P[0]

T = int(input())
for _ in range(T):
	N = int(input())
	idx = bs(PRIMES,0,len(PRIMES)-1,N)
	print(P[idx])
